# Como executar o projeto

## Frontend Angular
Pré-requisitos: 
- Node 16.14.2
- Angular 13.3.0

```bash
# clonar repositório
git clone git@github.com:Adyson-Lima-Programador/rodovtransport_web.git

# entrar na pasta rodovtransport_web
cd rodovtransport_web

# instalar dependencias com npm
npm install

# executar o projeto
npm start
```
